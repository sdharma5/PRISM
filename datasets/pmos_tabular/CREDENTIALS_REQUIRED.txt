PMOS Tabular Dataset (Kottarathil) - Kaggle Credentials Required
=================================================================
Dataset: Polycystic Ovary Syndrome (PMOS) Dataset
Author: Prasoon Kottarathil
Official Kaggle page: https://www.kaggle.com/datasets/prasoonkottarathil/polycystic-ovary-syndrome-pmos

Dataset details:
- Title: Polycystic Ovary Syndrome (PMOS)
- Contains: Clinical, biochemical, menstrual, symptom, and ultrasound-derived variables
- Patient-level data from 10 different hospitals across Kerala, India
- Two files: PMOS_infertility.csv (541 rows, ~44 cols, without infertility columns)
              PMOS_data_without_infertility.csv (541 rows, various features)
- Both contain a 'PMOS (Y/N)' label column
- License: CC0: Public Domain (as listed on Kaggle)
- The full clinical dataset includes: age, BMI, cycle regularity, AMH, FSH, LH,
  TSH, prolactin, antral follicle count (AFC), endometrium thickness,
  skin darkening, hair growth, weight gain, pimples, eating fast food, etc.

Manual access steps:
1. Create a Kaggle account at https://www.kaggle.com/
2. Go to https://www.kaggle.com/account → API → Create New Token
   This downloads kaggle.json. Place it at:
   ~/.kaggle/kaggle.json
   chmod 600 ~/.kaggle/kaggle.json
3. Install Kaggle CLI:
   pip install kaggle
4. Download the dataset:
   kaggle datasets download prasoonkottarathil/polycystic-ovary-syndrome-pmos \
     -p /weka/scratch/uchitra1/users/a_s/mit_hackathon/datasets/pmos_tabular/raw \
     --unzip
5. Verify files: PMOS_infertility.csv and PMOS_data_without_infertility.csv
